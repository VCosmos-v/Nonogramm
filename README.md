# Игра «Нонограммы» (C++)

## Консольная реализация логической головоломки Нонограммы (японские кроссворды).
Игрок решает задачу, закрашивая клетки по числовым подсказкам для строк и столбцов.

При запуске можно загрузить задачу из файла, создать свою и сохранить файл
Посмотреть поле загруженной задачи и начать игру

### После запуска игры доступны команды:

fill x y    — закрасить клетку

clear x y   — очистить клетку

solve       — применить решатель (одна строка за вызов)

check       — проверить решение

show        — показать поле

commands    — список команд

exit        — выход в меню


### Координаты:
x — номер столбца
y — номер строки
(нумерация начинается с 0)

### Формат файла задачи
Файл задачи — это решение, на основе которого автоматически строятся подсказки.

Файлы задач по умолчанию: field.txt, test.txt.
